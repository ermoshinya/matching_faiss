# matching_faiss

## Описание

**matching_faiss** — проект для поиска наиболее похожих товаров по векторным признакам с использованием библиотеки FAISS. Решение предназначено для задачи сопоставления товаров на основе эмбеддингов и оценки качества по метрике accuracy@5.

## Структура данных

- **base.csv** — анонимизированный набор товаров с уникальным id и вектором признаков (эмбеддингами).
- **train.csv** — обучающий датасет: id товара, вектор признаков, id наиболее похожего товара из базы.
- **validation.csv** — товары для валидации: id и вектор признаков.
- **validation_answer.csv** — правильные ответы для validation.csv.

## Задача

- Для каждого товара из validation.csv найти несколько наиболее похожих товаров из base.csv по эмбеддингам.
- Оценить качество поиска по метрике accuracy@5.

## Используемые библиотеки

- [FAISS](https://github.com/facebookresearch/faiss) — быстрый поиск по векторам.
- pandas, numpy — обработка данных.
- matplotlib, seaborn — визуализация.
- scikit-learn — препроцессинг, кластеризация, метрики.
- phik — корреляционный анализ.
